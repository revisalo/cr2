SolutionNumber

Select a sub-optimal MIP solution

Type: int
Default value: 0
Minimum value: 0
Maximum value: Infinity

When querying attribute Xn to retrieve an alternate MIP solution, this parameter determines which alternate solution is retrieved. The value of this parameter should be less than the value of the SolutionCount attribute.

Note: MIP only
