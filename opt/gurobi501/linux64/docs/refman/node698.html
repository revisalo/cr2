MIPGapAbs

Absolute MIP optimality gap

Type: double
Default value: 1e-10
Minimum value: 0
Maximum value: Infinity

The MIP solver will terminate (with an optimal result) when the absolute gap between the lower and upper objective bound is less than MIPGapAbs.

Note: MIP only
