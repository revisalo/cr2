QuadExpr.getVar2()

getVar2 ( i )

Retrieve the second variable for a single quadratic term of the quadratic expression.

Return value:

Second variable associated with the quadratic term at index i in the quadratic expression.

Example usage: 

  e = x * x + 2 * y * y + z
  print e.getVar2(1)
