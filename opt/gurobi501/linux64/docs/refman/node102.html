GRBterminate

void GRBterminate ( GRBmodel *model )

Generate a request to terminate the current optimization. This routine can be called at any time during an optimization. When the optimization stops, the Status attribute will be equal to GRB_INTERRUPTED.

Arguments:

model: The model to terminate.

Example usage: 

  if (time_to_quit)
    GRBterminate(model);
