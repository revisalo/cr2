GRBModel.GetVarByName()

Retrieve a variable from its name. If multiple variable have the same name, this routine chooses one arbitrarily.

GRBVar GetVarByName ( string name )

    Arguments:

    name: The name of the desired variable.

    Return value:

    The requested variable.
